# terraform-starter

Simple Terraform starter project.

## Files

- `versions.tf`: Terraform and provider version constraints.
- `main.tf`: Example resources.
- `variables.tf`: Input variables.
- `outputs.tf`: Output values.

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Notes

- This project uses the `random` provider for a minimal example.
- Update providers/resources to match your real infrastructure.
